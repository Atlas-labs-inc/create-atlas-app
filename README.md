# atlas-app
Node CLI tool to create boilerplate code for Atlas smart contracts

## Prerequisites
Docker, Node, and yarn

## Installation
```sh
git clone http://github.com/atlas-labs-inc/create-atlas-app
cd create-atlas-app
npm i -g
```
## Usage
1. Create new app
    ```sh
    atlas-app new
    ```
2. Install development dependencies 
    ```sh
    cd your-app && yarn
    ```
3. Deploy an Atlas app
    ```sh
    atlas-app deploy
    ```
