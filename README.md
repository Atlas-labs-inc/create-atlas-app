# atlas-app
Node CLI tool to create boilerplate code for Atlas smart contracts

## Prerequisites
Docker, Node, and yarn

## Installation
```sh
npm i atlas-app -g
```
## Usage
1. Create new app
    ```sh
    atlas-app new
    ```
2. Install development dependencies 
    ```sh
    cd your-app && yarn
    ```
3. Deploy an Atlas app
    ```sh
    atlas-app deploy
    ```
